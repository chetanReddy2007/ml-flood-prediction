Mission

Why FloodSense?

Floods are among the world's most destructive natural disasters. Traditional warning systems often react too late. FloodSense bridges that gap by offering an instant, ML-powered assessment based on current weather parameters, helping communities, engineers, and emergency teams make data-driven decisions before a flood event unfolds.

The system analyses five key meteorological indicators through a trained XGBoost classifier and outputs both a risk verdict and a probabilistic confidence score — giving responders the clarity they need.

Key Features

What It Offers

  • Real-time prediction — sub-second response with XGBoost inference
  • Confidence score — probabilistic output, not just a binary label
  • 5-feature model — annual & seasonal rainfall, visibility, temperature, humidity
  • Responsive dark UI — works on desktop, tablet & mobile
  • Production-ready — Gunicorn + Procfile for cloud deployment
  • Load-tested — Locust scripts included for scalability validation

Pipeline

How the Model Works

01
Synthetic Dataset Generation
5,000 weather samples are generated using realistic meteorological distributions. Labels are assigned via domain-expert rules (e.g., high annual rainfall + high humidity = High Risk) with 5% random label noise to prevent overfitting.
02
Pre-processing & Scaling
Missing values are imputed with column medians. A StandardScaler is fit only on training data to normalise all 5 features — preventing data leakage into the test set.
03
XGBoost Classifier Training
300 boosting rounds with learning rate 0.05, max depth 6, 80% column subsampling. The model is trained on 80% of the data and validated on 20%, achieving 95.1% accuracy.
04
Artefact Persistence
The trained model and scaler are saved as .pkl files using joblib. They are loaded once at Flask startup to minimise per-request latency.
05
Real-time Inference
User inputs are scaled with the saved scaler and passed to the model. predict_proba() returns the probability vector — giving a confidence % alongside the binary verdict.
